Features

Everything you need, nothing you don't.

🏠

Self-Hosted

Run on your own hardware. Docker Compose gets you running in minutes. Your data never leaves your network.

❤️

HealthKit Sync

The iOS companion app syncs 100+ data types from Apple Health — heart rate, sleep, workouts with GPS, ECG, and more.

📊

Correlation Explorer

Discover relationships between any two health metrics. See Pearson correlations with interactive scatter plots.

🤖

MCP for AI

Query your health data from Claude or any MCP-compatible assistant. 11 tools and 3 resources over stdio and SSE.

🔒

Tailscale Auth

Zero-config TLS and identity-based access via Tailscale. No passwords, no exposed ports.

💻

Open Source

MIT licensed. Go backend, React frontend, Swift iOS app. Single binary with embedded web UI.
